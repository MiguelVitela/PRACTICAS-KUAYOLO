Curso HTML

Esta es mi primer página HTML; Este nos sirve para escribir en negritas

Segundo párrafo; strong nos sirve para resaltar que un párrafo es importante, se ve igual que las negritas pero se le dá mayor importancia

Tercer párrafo; i nos sirve para escribir en italika o cursiva

Cuarto párrafo; em nos sirve para enfatizar un párrafo, visualmente se ve como la italika pero al igual que el strong se enfatiza más

Qinto párrafo; small nos sirve para mostrar un texto normal pero más pequeño

Sexto párrafo; mark nos sirvepara escribir un texto subrayado

Séptimo párrafo; del nos sirve para escribir un texto que se eliminará este aparecerá xon una línea horizontal enmedio

Octavo párrafo; ins nos sirve para mostrar un texto insertado, este se visualizará como un texto subrayado con una linea horizontal por debajo.

Noveno párrafo; sub se utiliza cuando queremos expresar un subíndice como por ejemplo con la fórmula del agua: H2O

Décimo párrafo: sup se utiliza cuando queremos expresar un superindice como por ejemplo: H2O


Usando los formatos de texto de HTML le damos un mayor sentido al texto que estamos agregando a nuestro documento y auqnue visualmente puede dar resultados es más importante el sentido dado a la información que el resultado visual.
